# Quality Checklist

Before finalizing any deck, verify:

## Structure
- [ ] Every title is an assertion (complete sentence with a claim)
- [ ] One idea per slide (no "and also")
- [ ] Conclusion/recommendation appears in first 3 slides

## Flow
- [ ] Transitions are explicit between sections
- [ ] Consistent visual language throughout

## Visuals
- [ ] Figures are simplified and highlighted
- [ ] Key data points are emphasized, not buried

## Accessibility
- [ ] Audience can understand without presenter (if async viewing)
- [ ] Text is readable at presentation scale
